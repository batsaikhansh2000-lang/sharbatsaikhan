# 🌊 Бохир усны гадна шугамын суулгалтын гүн тооцоолуур

Энэхүү вэб апп нь бохир усны гадна шугам хоолойн худгуудын хоорондох суулгалтын гүн болон Инверт (хоолойн ёроолын) түвшнийг автоматаар тооцоолох зориулалттай.

## ✨ Гол онцлогууд

* **Автомат налуугийн тооцоо:** Газрын гадаргуугийн налууг худгуудын хоорондох зайд үндэслэн автоматаар бодно.
* **Ухаалаг логик:** Хэрэв газрын налуу нь заасан минимумаас бага байвал шугамын ажиллагааг хангах үүднээс хамгийн бага налууг (0.007) автоматаар авч ашиглана.
* **Гүний хяналт:** Суулгалтын гүн нь стандарт хэмжээнд (0.7м - 6.0м) байгаа эсэхийг өнгөөр ялгаж анхааруулна.
* **Профайл диаграмм:** Тооцоолсон үр дүнг график хэлбэрээр шууд харуулна.

## 📐 Тооцооллын аргачлал

1.  **Инверт түвшин:** Invert(n) = Invert(n-1) - (Зай × Налуу)
2.  **Суулгалтын гүн:** Гүн(n) = Газрын ОТМ(n) - Инверт(n)
